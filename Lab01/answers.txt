- If the user choose cancer, the result return is "You have chosen No".
- To customize the options to users, we replace JOptionPane.showConfirmDialog to JOptionPane.showOptionDialog. 
For example in order to switch to "I do" and "I don't", we have the code:
Object[] options = {"I do",
                "I don't"};
int option = JOptionPane.showOptionDialog(null,
"Do you want to change the first class ticket?",
"Ticket question",
JOptionPane.YES_NO_OPTION,
JOptionPane.QUESTION_MESSAGE,
null,     
options,  
options[0]);
